# 如何添加项目到作品集

## 快速指南

### 步骤 1：编辑项目数据文件

打开 `src/data/projects.ts` 文件，添加或修改项目信息。

### 步骤 2：填写项目信息

每个项目包含以下字段：

```typescript
{
  id: 1,                    // 项目 ID（唯一）
  name: '项目名称',          // 项目标题
  description: '项目描述',   // 项目简介
  techStack: ['技术1', '技术2'],  // 使用的技术栈
  image: '/screenshot.jpg',  // 项目截图（可选）
  link: 'https://...',      // 项目在线地址
  githubLink: 'https://github.com/...',  // GitHub 仓库地址（可选）
}
```

### 步骤 3：添加项目截图（可选但推荐）

1. **截取项目截图**
   - 访问你的项目网站
   - 按 `F12` 打开开发者工具
   - 按 `Ctrl + Shift + P`（Mac: `Cmd + Shift + P`）
   - 输入 "screenshot"
   - 选择 "Capture full size screenshot"

2. **保存截图**
   - 将截图保存为 `markdown-editor.jpg` 或 `markdown-editor.png`
   - 放到项目的 `public` 目录下

3. **更新图片路径**
   ```typescript
   image: '/markdown-editor.jpg',
   ```

### 步骤 4：保存并测试

1. **保存文件**
   - 保存 `src/data/projects.ts`

2. **本地测试**
   ```bash
   npm run dev
   ```
   - 访问 http://localhost:5173/
   - 滚动到"项目展示"部分
   - 检查你的项目是否正确显示

3. **部署到 Vercel**
   ```bash
   git add .
   git commit -m "添加 Markdown 编辑器项目"
   git push
   ```
   - Vercel 会自动检测并重新部署

```typescript

    id: 1,
    name: 'Markdown 编辑器',
```

## 获取 GitHub 仓库地址

1. 访问你的 GitHub 仓库
2. 复制浏览器地址栏的 URL
3. 格式通常是：`https://github.com/用户名/仓库名`

如果项目还没有 GitHub 仓库：
1. 在 GitHub 创建新仓库
2. 推送代码到仓库
3. 复制仓库地址

## 常见问题

### Q1: 没有项目截图怎么办？

**方案 1：使用占位图**
- 不设置 `image` 字段
- 组件会自动显示渐变色占位图

**方案 2：使用在线截图工具**
- 访问 [screely.com](https://screely.com/)
- 上传项目截图
- 生成美化后的图片

### Q2: 如何调整项目顺序？

在数组中调整项目的位置即可，最上面的项目会最先显示。

### Q3: 可以添加多少个项目？

没有限制，但建议：
- 展示 3-6 个最好的项目
- 质量比数量更重要

### Q4: 项目链接失效了怎么办？

及时更新 `link` 字段，或者移除该项目。
