# 如何查看项目

## ❌ 错误方式

**不要直接双击打开 `index.html` 文件！**

直接打开 HTML 文件会看到空白页面，因为：
- 浏览器无法直接执行 TypeScript/JSX 代码
- 需要开发服务器来编译和运行代码
- React 应用需要构建过程

## ✅ 正确方式

### 方法 1：启动开发服务器（推荐）

这是开发时最常用的方法，支持热更新。

#### 步骤：

1. **打开终端/命令行**
   - Windows: 按 `Win + R`，输入 `cmd` 或 `powershell`
   - Mac: 打开"终端"应用
   - 或者在 VS Code 中按 `` Ctrl + ` ``（反引号）打开终端

2. **进入项目目录**
   ```bash
   cd "d:\kf\personal website"
   ```

3. **安装依赖**（如果还没安装）
   ```bash
   npm install
   ```
   
   这会安装所有需要的依赖包，可能需要几分钟。

4. **启动开发服务器**
   ```bash
   npm run dev
   ```

5. **查看项目**
   - 终端会显示类似这样的信息：
     ```
     VITE v5.0.0  ready in 500 ms
     
     ➜  Local:   http://localhost:5173/
     ➜  Network: use --host to expose
     ```
   - 打开浏览器，访问：**http://localhost:5173/**
   - 现在你应该能看到完整的网站了！

6. **停止服务器**
   - 在终端中按 `Ctrl + C` 停止服务器

---

### 方法 2：构建后预览

如果你想查看生产环境的构建结果：

1. **构建项目**
   ```bash
   npm run build
   ```
   
   这会创建一个 `dist` 目录，包含优化后的文件。

2. **预览构建结果**
   ```bash
   npm run preview
   ```

3. **查看项目**
   - 访问终端显示的 URL（通常是 `http://localhost:4173/`）

## 常见问题

### Q1: 运行 `npm install` 时出错

**可能原因：**
- Node.js 未安装
- npm 版本过旧

**解决方法：**
1. 安装 Node.js（推荐 18+ 版本）
   - 访问 [nodejs.org](https://nodejs.org/)
   - 下载并安装 LTS 版本
2. 验证安装：
   ```bash
   node --version
   npm --version
   ```

### Q2: 运行 `npm run dev` 时端口被占用

**错误信息：**
```
Port 5173 is in use
```

**解决方法：**
1. 关闭占用端口的程序
2. 或修改端口（在 `vite.config.ts` 中）

### Q3: 页面显示空白或报错

**检查：**
1. 查看浏览器控制台（F12）的错误信息
2. 检查终端是否有错误
3. 确保所有依赖已正确安装

### Q4: 修改代码后页面没有更新

**解决方法：**
- Vite 开发服务器支持热更新，修改代码后应该自动刷新
- 如果没有自动刷新，手动刷新浏览器（F5）

## 开发工作流

### 日常开发流程：

1. **启动开发服务器**
   ```bash
   npm run dev
   ```

2. **在浏览器中打开**
   - http://localhost:5173/

3. **编辑代码**
   - 修改 `src/` 目录下的文件
   - 保存后自动刷新

4. **查看效果**
   - 浏览器会自动更新显示最新内容

5. **停止服务器**
   - 按 `Ctrl + C`
